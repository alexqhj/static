Prompt

Set up Windows toast notifications for when you need my attention. Do the following steps:

1. Install wsl-notify-send:
   - mkdir -p ~/.local/bin
   - Download the amd64 zip from https://github.com/stuartleeks/wsl-notify-send/releases/latest using the GitHub API to find the correct asset URL (it's a zip, not a bare exe)
   - Extract wsl-notify-send.exe to ~/.local/bin/
   - chmod +x it
   - Verify ~/.local/bin is in PATH, if not add it to ~/.bashrc or ~/.zshrc

2. Test it works by running: wsl-notify-send.exe --category "Claude Code" "Test notification"

3. Add a Notification hook to ~/.claude/settings.json:
   {
     "hooks": {
       "Notification": [
         {
           "matcher": "",
           "hooks": [
             {
               "type": "command",
               "command": "wsl-notify-send.exe --category 'Claude Code' 'Claude needs your attention'"
             }
           ]
         }
       ]
     }
   }
   Merge this into the existing settings.json - don't overwrite other settings.

4. Verify the hook works by asking me a question using AskUserQuestion.

After setup

Customize the notification sound and behavior in Windows Settings > System > Notifications — look for "wsl-notify-send" in the app list.

Tip: The notification fires whenever Claude is waiting for your input — permission prompts, questions, idle state, etc. You can switch to another window and you'll get pinged when Claude needs you.
